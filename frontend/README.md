## Getting Started

First, Install the dependencies ( make sure u have nodejs and npm installed ):
```bash
npm install
```


Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
